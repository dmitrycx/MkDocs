# User Limits on Requests

Intrigma allows you to define **limits and rules** that control how many requests users can make, and what kinds of requests are allowed.

---

## Types of Limits

- Maximum requests per user per period
- Max days off per week or month
- Shift-specific limits (e.g., no more than 2 day-off requests on weekends)
- Limits by shift group or user group
