# Edit a Request

Make changes to an existing request on behalf of a user.

## Steps

1. Select the **User** from either:
    - The list of Users on the left side of the Dashboard, or
    - The list of Users in the Requests window

2. Select the request from the list to view its details at the bottom

3. Make your changes:

    | Field | Options |
    |-------|---------|
    | Status | Change between `Approved`, `Denied`, or `Unreviewed` |
    | Soft request | Check/uncheck to toggle soft request |
    | Note | Add or edit the request note |

!!! warning
    All edits are saved immediately without confirmation.
